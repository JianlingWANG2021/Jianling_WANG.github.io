High-Performance Computing (HPC)

HPC enables solving computationally intensive problems by using parallel computing frameworks and supercomputers. My research involves:

Large scale Nbody-hydrodynamical code development

To study the effect of hydrodynamical process in the gas during the galaxy formation and evolution, we have developed simulation code by implementing new physical modular to dealing with gas, e.g. star formation, cooling, feedback. These physical processes are implemented into two public simulation code, GIZIMO & GADGET-2 for cosmological and galaxy formation, which are written in C and MPI for massive-parallel calculations.

Parallel computing with MPI and CUDA

I wrote codes to derive stellar physical parameters (distance, stellar age, mass, extinction, effective temperature, metallicity, logg) for individual stars observed by LAMOST, which are important parameters for the Astronomers use LAMOST data for scientific studies. I used the Bayesian theorem plus prior galactic knowledge to derive these stellar parameters. The code was written in C language with multidimensional integration, which is impossible to run with a single computer for a few million stars. To solve this problem, I developed this code with MPI to use massively parallel computing. I also developed a CUDA version of the code with GPU computing, a single NVIDIA graphics device can achieve high performance.

Deep Learning in Scientific Research

Deep learning has revolutionized how we process complex data. My contributions include:

Deep learning in neural field and style transfering for 3D

Applying style transfer to a full 3D environment is a chal- lenging task that has seen many developments since the advent of neural rendering. 3D Gaussian splatting (3DGS) has recently pushed further many limits of neural rendering in terms of training speed and reconstruction quality. This work introduces SGSST: Scaling Gaussian Splatting Style Transfer, an optimization-based method to apply style transfer to pretrained 3DGS scenes. We demonstrate that a new multiscale loss based on global neural statistics, that we name SOS for Simultaneously Optimized Scales, enables style transfer to ultra-high resolution 3D scenes. Not only SGSST pioneers 3D scene style transfer at such high image resolutions, it also produces superior visual quality as assessed by thorough qualitative, quantitative and perceptual comparisons.

Machine learning applications in astrophysics

I used ML to measure stellar metallicity with photometric data. Without time-consuming spectra observation, I can acheive accurate metallicity for faint stars.

Combining ML, Markov chain Monte Carlo (MCMC), and N-body simulations, I developed a technique to improved the orbital trajector of multi-galaxies merger history.
